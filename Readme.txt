Project Title: Attribute-based File Sharing System

Project Description:
Create a file sharing platform with attribute-based access
control.
Key functionalities:
administrator can register users and provide each user a
secret key based on the user’s attributes
Each registered user can login the platform to upload files
that will be encrypted based on an access policy specified
the uploader
Each registered user can only open files whose associated
access policies are satisfied by the user’s attributes

Technology Description:
A Windows/Mac application or a web-based system
Cryptographic libraries. e.g., the Pairing-based
cryptographic library
